# nico3d_feed
ニコニ立体の新着をatomフィードにします。

## Install
For Ubuntu 16.04 and apache

```sh
$ cd /var/www 
$ sudo git clone https://github.com/sanadan/nico3d_feed.git
$ cd nico3d_feed
$ sudo ./setup
```

設置したサーバーの nico3d_feed にアクセスすると、フィードが出力されます。

フィードは約1時間毎に更新されます。
